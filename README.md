# Учебный проект (C++)

Проект показывает, как разнести каждое задание в отдельный `.cpp`/`.hpp`, но запускать их из одного `main.cpp`. Каждый модуль делает простую консольную демонстрацию.

## Структура
- `src/main.cpp` — меню для запуска отдельных задач.
- `src/task_*.cpp` + `include/tasks.hpp` — реализации заданий.
- `data/set_data.txt` — пример входных данных для задания с множеством.
- `TODO.txt` — оригинальное ТЗ.

## Сборка
Через CMake:
```bash
cmake -S . -B build
cmake --build build
./build/student_tasks
```
Или напрямую:
```bash
g++ -std=c++17 src/*.cpp -I include -o student_tasks
./student_tasks
```

## Как пользоваться
1. Запустите бинарник и введите номер задания.
2. Следуйте подсказкам:
   - **1** — вычисление выражения со скобками, `+ - *`, унарный минус.
   - **2** — загрузка множества из файла (по умолчанию `data/set_data.txt`), команды `SETADD`, `SETDEL`, `SET_AT`.
   - **3** — разбиение множества на подмножества с заданной суммой (вводите элементы, заканчивая `0`).
   - **4** — вывод всех подмножеств массива.
   - **5** — построение BST по вводимой последовательности (до `0`) и проверка АВЛ-сбалансированности (`YES/NO`).
   - **6** — маленький эксперимент: вставка `n` элементов, `m` поисков; сравнение цепочек и открытой адресации (двойное хеширование).
   - **7** — подсчёт количества потомков для каждой вершины родословного дерева; вывод в лексикографическом порядке.

Код можно модифицировать/дополнять под собственные варианты, каждый модуль изолирован и подключается через общее меню.
